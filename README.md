# Math Games Web App

A comprehensive collection of educational math games and puzzles built with HTML, CSS, and JavaScript.

## Features

### Game Categories

**Math Games**
- Addition - Basic arithmetic with scaling difficulty
- Subtraction - Number subtraction challenges  
- Multiplication - Times tables practice
- Division - Perfect division problems

**Math Puzzles**
- Missing Number - Find missing values in equations
- Number Pattern - Identify mathematical sequences
- Logic Puzzle - Word problems requiring mathematical reasoning
- Simple Algebra - Solve for x with helpful hints

**Logic Games**
- Memory Match - Card matching with emoji symbols
- Number Sequence - Pattern recognition challenges

### Key Features
- **Progressive Leveling System** - Difficulty increases as you advance
- **Score Tracking** - Keep track of your progress
- **Visual Progress Bar** - See advancement toward next level
- **Responsive Design** - Works on desktop and mobile devices
- **Keyboard Support** - Press Enter to submit answers
- **Educational Hints** - Helpful guidance for puzzle solving

## How to Play

1. Open `index.html` in your web browser
2. Choose a game category and specific game type
3. Answer questions to earn points and advance levels
4. Use the "End Game" button to return to the main menu anytime

## Technical Details

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **No Dependencies** - Pure vanilla JavaScript
- **Responsive Design** - CSS Grid and Flexbox
- **Local Storage** - No server required
